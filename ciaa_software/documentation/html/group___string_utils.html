Conversión de número en punto fijo a string.

Convierte el número en punto fijo pasado por argumentos en una cadena de caracteres representativa. Esta cadena se escribe en un buffer.

Parámetros
[in]valorNúmero en punto fijo a convertir a string.
[in]expDetermina la posición del separador de parte entera y decimal. Equivale a dividir valor por 10^(exp).
[in]minCantDigitosMínima cantidad de dígitos a devolver. Si no se alcanza esta cantidad, se agregan ceros a la izquierda.
[out]strBuffer donde se guardará la cadena resultante. Debe tener espacio suficiente para todos los números, un carácter separador de parte entera/decimal y el carácter nulo.
Devuelve
Posición en el buffer luego de escribir la cadena. Apunta al carácter nulo.

The strlcpy() function copy strings It is designed to be safer, more consistent, and less error prone replacements for strncpy. Unlike these function, strlcpy() take the full size of the buffer (not just the length) and guarantee to NUL-terminate the result (as long as size is larger than 0). Note that a byte for the NUL should be included in size. Also note that strlcpy() only operate on true 'C' strings. This means that src must be NUL-terminated.

The strlcpy() function copies up to size - 1 characters from the NUL-terminated string src to dst, NUL-terminating the result.

Ver también
https://www.freebsd.org/cgi/man.cgi?query=strlcpy&sektion=3
Parámetros
[out]destDestination buffer.
[in]srcSource string.
[in]sizeSize of the buffer.
Devuelve
The strlcpy() function return the total length of the string it tried to create, that means the length of src.
