Descripción detallada

Interfaz unificada para el manejo de los parsers.

Provee una interfaz común y declara ciertos tipos utilizados para crear y manipular parsers.

Un parser es un módulo de software que cumple con ciertas especificaciones y que tiene como finalidad recibir caracteres, uno a uno, y decidir si cumplen con algún patrón o regla dados. A su vez, existe la posibilidad que al detectar el patrón, también se extraigan datos de éste y sean devueltos al usuario como resultados del parseo.

Documentación de las enumeraciones

enum ParserStatus

Estados posibles de un parser.

Valores de enumeraciones
STATUS_UNINITIALIZED 

Todavía no se ha llamado parser_init() en el parser.

STATUS_INITIALIZED 

Se ha llamado parser_init(), pero no se hizo nada más.

STATUS_INCOMPLETE 

El último carácter ingresado mediante parser_tryMatch() es válido pero no completa el patrón.

STATUS_NOT_MATCHES 

El último carácter ingresado mediante parser_tryMatch() no puede formar parte del patrón.

STATUS_COMPLETE 

El último carácter ingresado mediante parser_tryMatch() ha completado exitosamente el patrón.

enum ParserType

Valores reservados para el tipo de parser. Si se agrega un nuevo parser, es conveniente agregar el tipo en este enumerativo.

Si por algún razón no importa el tipo, establecerlo en un valor distinto a los reservados.

De-inicializa un parser.

Esta función pone al parser en un estado inválido, por lo cual si se lo quiere volver a usar, deberá ser inicializado de nuevo con parser_init().

void* parser_getResults ( const Parser *  parser)

Obtiene el resultado del parseo.

Llamar sólo si el parser tiene como estado STATUS_COMPLETE, de lo contrario, los valores que retorne pueden ser incorrectos.

Parámetros

Devuelve

int32_t parser_init ( Parser *  parser)

Inicializa el parser especificado.

Establece los valores iniciales para cada parser. Si se quiere reiniciar su estado, se lo puede hacer llamando de nuevo esta función sobre el mismo.

Es obligatorio llamar esta función antes de empezar a usar un parser.

Parámetros
[in]parserPuntero al parser a inicializar.
Devuelve
Retorna un valor positivo si la inicialización fue correcta, de lo contrario el valor será negativo.
ParserStatus parser_tryMatch ( Parser *  parser,
uint8_t  newChar 
)

Ingresa un carácter para intentar formar el patrón.

Un nuevo carácter es ingresado para analizar si forma el patrón buscado en conjunto con los caracteres anteriormente ingresados.

Esta operación siempre modifica el estado del parser.

Parámetros
[in]parserPuntero al parser sobre el cual se desea ver si el carácter pertenece al patrón.
[in]newCharCarácter a ingresar al parser.
Devuelve
Estado del parser luego de haber ingresado el carácter.
